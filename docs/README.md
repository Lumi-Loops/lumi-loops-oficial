## 🛠️ Stack Tecnológico

- **Framework**: Next.js 15.5.6 con App Router
- **UI**: React 19.1.0
- **Lenguaje**: TypeScript 5
- **Estilos**: Tailwind CSS 4
- **Package Manager**: Bun
- **Build Tool**: Turbopack
